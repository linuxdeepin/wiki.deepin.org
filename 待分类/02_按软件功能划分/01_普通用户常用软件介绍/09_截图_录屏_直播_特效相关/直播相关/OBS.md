# OBS安装
## 前言
OBS Studio（Open Broadcaster Software）是一个跨平台的开源的直播录屏软件，是Linux系统上最受欢迎的录屏软件
## 安装
    	sudo apt update
    	sudo apt install ffmpeg obs-studio
## 使用
打开软件按照向导配置，在录屏时需要添加源，直播需要配置直播服务器
