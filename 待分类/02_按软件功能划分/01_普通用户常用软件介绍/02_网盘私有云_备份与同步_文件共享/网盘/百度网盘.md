## 简介
百度网盘是一款省心、好用的超级云存储产品，已为超过7亿用户提供云服务。空间超大，支持多类型文件的备份、分享、查看和处理。

## 安装
```
sudo apt install com.baidu.baidunetdisk
```

## 卸载
```
sudo apt remove --purge com.baidu.baidunetdisk
```
