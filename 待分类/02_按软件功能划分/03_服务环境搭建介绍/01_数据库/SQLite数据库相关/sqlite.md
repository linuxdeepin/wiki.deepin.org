# 简介

SQLite是遵守ACID的关联式数据库管理系统，它包含在一个相对小的C库中。

## 安装

`sudo apt-get install sqlite`			安装SQLite版
`sudo apt-get install sqlite3`		安装SQLite3版

## 卸载

`sudo apt-get remove sqlite`			卸载SQLite版
`sudo apt-get remove sqlite3`			卸载SQLite3版
