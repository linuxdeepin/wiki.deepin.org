# 简介

 unixODBC 是一个来连接数据库的组件，说白了它提供了一种标准的API，即应用程序编程接口的方法来访问DBMS(Database Management System)。

## 安装

`sudo apt-get install unixodbc`
## 卸载

`sudo apt-get remove unixodbc`
