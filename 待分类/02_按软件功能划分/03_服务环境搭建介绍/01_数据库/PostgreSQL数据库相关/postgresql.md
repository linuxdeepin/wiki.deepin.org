# 简介

PostgreSQL 是一个对象-关系数据库服务器(ORDBMS)。

## 安装

`sudo apt-get install postgresql-9.6`
postgresql有多个版本，可以手工指定包名postgresql-x.y（x为大版本号）来安装对应的版本。
## 卸载

`sudo apt-get remove postgresql-9.6`
