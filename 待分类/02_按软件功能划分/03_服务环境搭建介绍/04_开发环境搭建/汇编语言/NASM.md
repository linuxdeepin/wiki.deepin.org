## 简介
Netwide Assembler（简称 NASM）是一款基于英特尔 x86 架构的汇编与反汇编工具。它可以用来编写16位、32位（IA-32）和64位（x86-64）的程序。 NASM被认为是Linux平台上最受欢迎的汇编工具之一。

## 安装
```
sudo apt install nasm
```

## 卸载
```
sudo apt remove --purge nasm
```

## 用法
```
nasm [-@ response_file] [options...] [--] filename
nasm -v (or --v)
```
