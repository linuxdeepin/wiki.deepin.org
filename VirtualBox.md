## 简介

Oracle VM VirtualBox是一款开源的虚拟机软件，它可以虚拟化安装Windows、Mac OSX、Linux、OpenBSD、Solaris等系统。

## 安装

`sudo apt-get install virtualbox`

## 卸载

`sudo apt-get remove pool/main/v/virtualbox/`
