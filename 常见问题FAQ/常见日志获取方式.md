## 启动器日志
**日志获取路径**
```bash
~/.cache/deepin/dde-launcher/dde-launcher.log 
```
**同时提供应用版本信息：**
```bash
dpkg -l > Desktop/version
```

## 系统日志

- 系统普通日志

```shell
# 使用sudo权限查看日志文件最后10行
sudo less /var/log/messages
```

- 系统启动日志，包括启动引导时的日志

```shell
sudo dmesg | less
```
