# 美化dock栏

### 说明
Plank 是一个用 Vala 语言编写的、 简单的、轻量级的 dock 工具，它也是 Docky  3.0.0 上面的一种底层技术，致力于提供所有的核心功能；同时它允许 Docky 扩展 Docklets，允许用户自定义屏幕位置、主题、图标大小等功能，还支持自定义隐藏模式。

### 安装
```
sudo apt install plank
```
