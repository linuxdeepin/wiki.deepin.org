## 简介
uTools，新一代效率工具平台
自由组合插件应用，打造专属你的趁手工具集
在任何界面下通过快捷键「Alt + 空格」呼出 uTools，或是点击鼠标中键呼出超级面板
一个软件该有的样子：即用即走，无打扰，不中断你当下的工作

## 安装
```
sudo apt install u.utools.utools
```
## 卸载
```
sudo apt remove --purge u.utools.utools
```
