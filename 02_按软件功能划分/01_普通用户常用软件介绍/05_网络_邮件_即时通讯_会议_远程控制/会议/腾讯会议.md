## 简介
腾讯会议是腾讯基于21年音视频通讯经验、并依托于腾讯云全球化网络部署出品的简单易用、高清流畅、安全可靠的云会议协作平台。界面清爽，操作简单，在线文档协作、小程序入会、会管会控、屏幕共享等功能一应俱全，让您随时随地、秒级入会，提升会议效率。

## 安装
```
sudo apt install com.qq.wemeet
```

## 卸载
```
sudo apt remove --purge com.qq.wemeet
```
