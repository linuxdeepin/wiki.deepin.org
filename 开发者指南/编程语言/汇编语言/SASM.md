## 简介
SASM（SimpleASM 的缩写）是一个免费的开源跨平台集成开发环境（IDE），用于NASM、MASM、GAS和FASM汇编语言。它具有语法高亮显示功能，并包含一个调试器。

![sasm.png](/sasm.png)

## 安装
```
sudo apt install sasm
```

## 卸载
```
sudo apt remove --purge sasm
```
