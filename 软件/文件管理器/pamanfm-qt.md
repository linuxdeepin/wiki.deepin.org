## 简介

PCMan File Manager Qt ( PCManFM-Qt) 是 LXQT 桌面默认的文件管理器，PCMan File Manager的QT移植版，您可以使用它来浏览硬盘、U 盘、SD 卡和其他存储设备中的内容，也可以方便快捷地创建、移动、删除文件和文件夹。

## 安装

`sudo apt-get install pcmanfm-qt`

## 卸载

`sudo apt-get remove pcmanfm-qt`
