## 简介

Dolphin 是 KDE 的文件管理器，您可以使用它来浏览硬盘、U 盘、SD 卡和其他存储设备中的内容，也可以方便快捷地创建、移动、删除文件和文件夹。

## 安装

`sudo apt-get install dolphin`

## 卸载

`sudo apt-get remove dolphin`
