## 简介

yEd Graph Editor是一款流程图绘制工具，可以很方便的创建流程图， UML 图表，意向图以及其它类型的图表，图形和网络结构图。内置丰富的素材和工具，支持自动布局功能，可以轻松导入XML、GEDCOM、GML等格式文件，也可以将内容保存，打印或者导出为多种主流格式。

## 安装

`sudo apt-get install yed`

## 卸载

`sudo apt-get remove yed`
