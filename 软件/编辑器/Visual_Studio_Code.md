## 简介

Visual Studio Code是一款轻量级代码编辑器，它具有语法高亮、可定制的热键绑定、括号匹配、代码片段收集、代码调试、自定义语言和主题、支持扩展/插件等功能。

## 安装

`sudo apt-get install vscode`

## 卸载

`sudo apt-get remove vscode`
